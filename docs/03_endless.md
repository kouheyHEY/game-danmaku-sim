# 方針転換2：極限までシンプルなエンドレス回避

> 既存の `World`/`step()`/弾幕ライブラリ/被弾-復帰演出はそのまま再利用。
> ローグライトの「敵列＋3択リワード」は撤去する。

## コアループ（ワンフィンガー）

1. 起動 → **「Tap to Start」だけ**表示。タップで開始。
2. 画面上部から**弾が降り続ける**のを、**ドラッグでひたすら避ける**だけ。
3. 自機は**自動発射**。攻撃操作は不要。
4. **たまにボス出現**（動く標的）。自弾で撃破すると **HP+1回復** ＆ **弾幕をランダム強化**（弾速/弾数/連射/威力/拡散）。
5. 被弾で HP-1（その場で爆発→画面下から点滅復帰）。HP0 で **ゲームオーバー → スコア表示 → タップで再開**。

## スコア

**避けた弾の数**（画面右上）。＝ 敵弾が画面外に消えた回数（`world.dodged`）。

## ドメインの最小追加

- `World.emitterPos`：敵弾の発生源（上部中央固定）。敵(=ボス)の有無に依らず**常に雨が降る**ようにする。
- `World.dodged`：画面外に出た敵弾のカウント（＝スコア）。
- 強化はランダム自動適用（選択UIなし）。

## レイヤ

```
Session（run/session.ts） title / playing / gameover の状態機械
  ├ ambientRain(level)     全幅ランダムに降る雨（levelで濃く・速く）
  ├ makeBoss(level)        たまに出る動く標的（HPあり）
  └ randomWeaponUpgrade    撃破時にランダムで武器強化
World/step（domain）        無改修に近い形で流用（emitterPos/dodged のみ追加）
SessionRenderer（render）   弾/自機/ボスHP＋右上スコア＋Tap to Start／GameOver
```

反転(①〜④)・敵列・3択は撤去（コードは履歴に残る）。まずは「触ってすぐ・ずっと避ける」体験に一本化する。
